Decentraland Worlds Content Server API (1.0.0)

Download OpenAPI specification:

License: Apache 2.0

API for Decentraland Worlds, a content server that lets you deploy isolated scenes which are accessible by the Decentraland explorer.

Worlds are scenes that exist outside the Genesis City boundaries and are attached to Decentraland NAMEs. The NAMEs can be used as realms to access your deployed scenes.

Note: This server is in ALPHA version.

Worlds Content Server
